# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Проверка конфигурации

Ваши данные уже добавлены в `config/config.yaml`:

- ✅ API ID и API Hash
- ✅ Bot Token и Chat ID
- ✅ Канал для скачивания

Если нужно добавить еще каналы, откройте `config/config.yaml` и добавьте их в список `channels`.

## Шаг 3: Первый запуск

```bash
python run.py
```

**При первом запуске:**

1. Введите номер телефона (с кодом страны, например: +79991234567)
2. Введите код подтверждения из Telegram
3. После этого сессия сохранится и код больше не понадобится

## Шаг 4: Что происходит

1. Бот подключается к Telegram
2. Сканирует указанный канал
3. Находит все видео с качеством 360p, 480p или 720p
4. Проверяет, какие файлы уже скачаны
5. Загружает новые файлы параллельно (5 файлов одновременно)
6. Сохраняет файлы в `downloads/[имя_канала]/`
7. Отправляет уведомление в Telegram о завершении

## Где найти скачанные файлы?

Файлы сохраняются в папке `downloads/`:

```
downloads/
└── Full Hentai 18+/
    ├── video1_720p.mp4
    ├── video2_480p.mp4
    └── ...
```

## Добавление новых каналов

Откройте `config/config.yaml` и добавьте каналы в список:

```yaml
channels:
  - "https://t.me/joinchat/AAAAAFZTTF99JzOWLZH74A"
  - "@another_channel" # Добавьте новый канал
  - -1001234567890 # Или по ID
```

## Настройка качества видео

В `config/config.yaml` можно изменить качества:

```yaml
download:
  video_qualities: [360, 480, 720] # Какие качества скачивать
```

## Настройка параллелизма

```yaml
download:
  max_concurrent: 5 # Количество одновременных загрузок (можно увеличить до 10)
```

## Проблемы?

1. **Ошибка авторизации**: Удалите папку `sessions/` и запустите снова
2. **Канал не найден**: Убедитесь, что у вас есть доступ к каналу
3. **Медленная загрузка**: Уменьшите `max_concurrent` или проверьте интернет

## Логи

Логи сохраняются в:

- Консоль (всегда видно)
- `logs/downloader.log` (файл)
